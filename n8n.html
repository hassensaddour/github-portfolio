🌤️ Daily Weather Email Workflow

This n8n workflow automatically fetches the daily weather forecast for Ghar El Melh every morning at 7:00 AM and sends a formatted weather report to your Gmail inbox.

Features:

  • 🌐 Uses open-meteo.com API for free weather data
  • 🧠 A Code node extracts today’s high and low temperatures
  • 📩 Sends the result via Gmail with a clean, readable message
  • ⏱ Scheduled to run daily (or on demand)
