Daily Weather Reporter

A personal utility workflow that fetches daily forecast data for Ghar El Melh and delivers a formatted briefing every morning at 7:00 AM.

>> Workflow_Specs:
  • API Integration: Fetches JSON data from open weather APIs.
  • Data Processing: Code node parses high/low temps and conditions.
  • Delivery: Formats HTML email and sends via Gmail SMTP.
  • Scheduling: CRON trigger set for daily execution.

AI Social Media Automation

An intelligent automation workflow that generates AI-powered social media posts, requests human approval, and automatically publishes them to multiple platforms. Built using n8n to orchestrate integrations between AI models, content databases, messaging systems, and social media APIs.

>> Workflow_Specs:
  • Content Management: Retrieves topics and descriptions from a Notion database acting as a content calendar.
  • AI Generation: Uses Gemini AI to generate engaging social media posts based on the topic.
  • Human Approval: Sends generated posts to Telegram for manual approval before publishing.
  • Media Handling: Downloads images from Google Drive for use in social media posts.
  • Multi-Platform Publishing: Automatically publishes approved posts to LinkedIn and Facebook.
  • Automation: Scheduled workflow executes daily and updates Notion status to avoid duplicate content.
